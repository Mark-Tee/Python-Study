### <span style="color:rgb(144,120,249)"> Python简介</span>

 <span style="color:rgb(144,120,249)">编程语言</span>是用来定义计算机程序的语言，用来向计算机发出指令

 <span style="color:rgb(144,120,249)">Python语言</span>是一种<span style="color:red">面向对象</span>的<span style="color:red">解释型</span>高级编程语言

 python是强类型的动态脚本语言


### <span style="color:rgb(144,120,249)">编译型和解释型语言的</span><span style="color:rgb(144,120,249)">区别</span>
1. 编译型语言编译后就可以在平台运行，解释型语言在运行期间才编译

2. 一般来说，编译型语言运行速度快

3. 解释型语言跨平台特性比编译型语言好

### <span style="color:rgb(144,120,249)">编写第一个程序</span>

```python
print("Hello World")
```

注意：

1. 输入错误：python中的符号都是要用英文模式下的
2. 缩进错误：print要定格写，否则就报错
3. 语法错误：一个print必须单独写一行，错误信息遇到Syntax说明语法有问题
4. 命名错误：字符串必须加上引号，单引号或双引号都可

### <span style="color:rgb(144,120,249)">注释</span>
注释可以放在任意位置，但是注释的内容不会被程序执行

### <span style="color:rgb(144,120,249)">输出函数print()</span>
print()函数可以输出字符串、数字、变量等内容

```python
print("Hello World") # 输出字符串
print(123) # 输出数字
name = "Alice"
print(name) # 输出变量
print("Hello", "World") # 输出多个内容，默认用空格分隔
print("Hello" + "World") # 输出多个内容，使用字符串连接符
```
### <span style="color:rgb(144,120,249)">输入函数input()</span>
input()函数可以从用户那里获取输入，返回一个字符串

```python
name = input("请输入你的名字：") # 提示用户输入
print("你好，" + name) # 输出用户输入的内容
age = input("请输入你的年龄：") # 获取用户输入的年龄
print("你今年" + age + "岁") # 输出用户输入的年龄
```
注意：input()函数返回的内容都是字符串，如果需要进行数值计算，需要将输入转换为相应的数据类型

```python
age = int(input("请输入你的年龄：")) # 获取用户输入的年龄并转换为整数
print("你今年" + str(age) + "岁") # 输出用户输入的年龄
``` ### <span style="color:rgb(144,120,249)">变量</span>
变量是用来存储数据的容器，可以在程序中使用变量来保存和  操作数据
```python
name = "Alice" # 定义一个字符串变量
age = 30 # 定义一个整数变量
height = 1.75 # 定义一个浮点数变量
print(name) # 输出变量的值
print(age)
print(height)
```
注意：
1. 变量名必须以字母或下划线开头，后面可以跟字母、数字或下划线
2. 变量名不能是Python的保留字，如if、for、while等
3. 变量名区分大小写，name和Name是两个不同的变量
4. 变量可以随时改变值，但必须先定义后使用
```python
name = "Alice" # 定义一个变量
print(name) # 输出变量的值
name = "Bob" # 重新赋值变量
print(name) # 输出新的变量值
```
### <span style="color:rgb(144,120,249)">数据类型</span>
Python中常见的数据类型有：
1. 数字类型：整数（int）和浮点数（float）
2. 字符串类型（str）
3. 布尔类型（bool）：True和False
4. 列表类型（list）：用来存储一组有序的元素
5. 元组类型（tuple）：用来存储一组有序的元素，但不可修改
6. 字典类型（dict）：用来存储一组键值对
7. 集合类型（set）：用来存储一组无序的唯一元素
### <span style="color:rgb(144,120,249)">类型转换</span>
Python提供了内置的类型转换函数，可以将一种数据类型转换为另一种数据
```python
num_str = "123" # 定义一个字符串变量
num_int = int(num_str) # 将字符串转换为整数
print(num_int) # 输出整数
num_float = float(num_str) # 将字符串转换为浮点数
print(num_float) # 输出浮点数
```
注意：
1. 转换函数会尝试将输入转换为指定的类型，如果转换失败会抛出异常
2. 转换函数只能转换兼容的类型，如字符串"123"可以转换为整数，但字符串"abc"无法转换为整数
3. 转换函数可以用于变量之间的类型转换，如将整数转换为字符串
```python
num = 123 # 定义一个整数变量
num_str = str(num) # 将整数转换为字符串
print(num_str) # 输出字符串
``` 
### <span style="color:rgb(144,120,249)">运算符</span>
Python中常见的运算符有：
1. 算术运算符：+、-、*、/、%（取模）、**（幂运算）
2. 比较运算符：==、!=、>、<、>=、<=
3. 逻辑运算符：and、or、not
4. 位运算符：&（按位与）、|（按位或）、^（按位异或）、~（按位取反）、<<（左移）、>>（右移）
5. 赋值运算符：=、+=、-=、*=、/=、%=、**=等
6. 成员运算符：in、not in
7. 身份运算符：is、is not
```python
a = 10
b = 3
print(a + b) # 输出13
print(a - b) # 输出7
print(a * b) # 输出30
print(a / b) # 输出3.3333333333333335
print(a % b) # 输出1
print(a ** b) # 输出1000
print(a == b) # 输出False
print(a != b) # 输出True
print(a > b) # 输出True
print(a < b) # 输出False
print(a >= b) # 输出True
print(a <= b) # 输出False
print(a > 5 and b < 5) # 输出True
print(a > 5 or b > 5) # 输出True
print(not(a > 5)) # 输出False
print(a & b) # 输出2
print(a | b) # 输出11
print(a ^ b) # 输出9
print(~a) # 输出-11
print(a << 1) # 输出20
print(a >> 1) # 输出5
print(a in [10, 20, 30]) # 输出True
print(b not in [1, 2, 3]) # 输出True
print(a is 10) # 输出True
print(b is not 3) # 输出False
```
### <span style="color:rgb(144,120,249)">运算符优先级</span>
运算符优先级决定了在一个表达式中，哪些运算先执行
1. 括号：()、[]、{}
2. 指数运算：**
3. 正负号：+x、-x
4. 乘除运算：*、/、%、//
5. 加减运算：+、-
6. 位运算：&、|、^、~、<<、>>
7. 比较运算：==、!=、>、<、>=、<=
8. 逻辑运算：and、or、not
9. 赋值运算：=、+=、-=、*=、/=、%=
10. 成员运算：in、not in
11. 身份运算：is、is not
```python
result = 3 + 4 * 2 / (1 - 5) ** 2 ** 3
print(result) # 输出3.0001220703125
```
注意：
1. 括号可以改变运算的优先级，括号内的运算会先执行
2. 在没有括号的情况下，按照运算符优先级从高到低执行
3. 同一优先级的运算符按照从左到右的顺序执行
4. 赋值运算符的优先级最低，通常在表达式的最后执行
5. 逻辑运算符的优先级比比较运算符低，所以在表达式中比较运算会先执行
```python
result = 3 > 2 and 5 < 10
print(result) # 输出True
```
### <span style="color:rgb(144,120,249)">表达式和语句</span>
表达式是由一个或多个数字、变量、运算符和函数调用组成的组合，可以被求值为一个结果
语句是由一个或多个表达式组成的代码行，执行时会产生某种效果，如赋值、输出、条件判断等
```python
x = 10 # 这是一个赋值语句
y = x + 5 # 这是一个赋值语句，右边是一个表达式
print(y) # 这是一个输出语句，括号内是一个表达式
```
注意：
1. 表达式可以作为语句的一部分，但语句不能作为表达式的一部分
2. 表达式可以返回一个值，而语句通常不返回值
3. 语句可以包含多个表达式，但表达式不能包含语句
4. 语句可以独立存在，而表达式必须在某个上下文中
```python
x = 10 # 这是一个赋值语句
y = x + 5 # 这是一个赋值语句，右边是一个表达式
print(y) # 这是一个输出语句，括号内是一个表达式
```
### <span style="color:rgb(144,120,249)">代码风格</span>
代码风格是指编写代码时遵循的一套规范和约定，可以提高代码的可读性和可维护性
1. 使用4个空格进行缩进，不要使用制表符（Tab）
2. 每行代码不超过79个字符
3. 使用空行分隔函数和类的定义，以及代码的不同部分
4. 使用有意义的变量名和函数名，避免使用单个字母或无意义的名称
5. 在运算符两侧添加空格，如a + b，而不是a+b
6. 在函数定义和调用时，参数之间使用空格分隔，如def func(a, b)而不是def func(a,b)
7. 在注释前添加一个空格，如# 这是一个注释，而不是#这是一个注释
8. 使用驼峰命名法（CamelCase）来命名类，如MyClass，而使用小写字母和下划线来命名函数和变量，如my_function和my_variable
9. 在文件开头添加模块级别的文档字符串（docstring），描述模块的功能和用途
```python
"""
这是一个示例模块，包含一些示例函数和变量
"""
class MyClass:
    """这是一个示例类，包含一些示例方法和属性"""
    
    def __init__(self, name):
        """初始化方法，接受一个name参数"""
        self.name = name # 定义一个实例属性
    
    def greet(self):
        """一个示例方法，输出问候语"""
        print("Hello, " + self.name + "!") # 输出问候语
def my_function(x, y):
    """这是一个示例函数，接受两个参数并返回它们的和"""
    return x + y # 返回参数的和
my_variable = 42 # 定义一个示例变量
```
注意：
1. 代码风格虽然不是强制性的，但遵循统一的风格可以提高代码的可读性和可维护性，尤其是在团队合作中
2. Python社区推荐使用PEP 8作为代码风格指南，可以参考PEP 8文档来了解更多细节和示例
3. 使用代码格式化工具如Black或autopep8可以自动调整代码风格
4. 在编写代码时，保持一致的风格和格式可以让代码更易于阅读和理解，也有助于减少错误和提高效率
