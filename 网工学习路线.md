## 推荐推进节奏

```
已完成的          下一步(1-2周)          实战(同步推进)
┌──────────┐     ┌────────────┐     ┌──────────────────┐
│ 基础语法 │ →  │ pip + venv │ →  │ netmiko 批量巡检  │
│ 数据结构 │     │ re 正则    │     │ 自动备份 running  │
│ 函数工具 │     │ JSON/CSV   │     │ 端口状态报表      │
│ 文件读写 │     │ datetime   │     │ 流量数据分析      │
└──────────┘     └────────────┘     └──────────────────┘
                        │
                        └── 学完立即实战，不堆积
```

## 核心原则

1. **够用就行**：第二阶段学完立刻进入实战，不要等所有高级知识学完才动手
2. **实战驱动**：每个知识点配一个网工场景练习（比如学完 re 就解析真实的 show 输出）
3. **库 > 语法**：网工瓶颈不在 Python 语法，在"知不知道那个库存在"
4. **先模仿再创造**：netmiko 官方示例 → 小改 → 大改 → 自己写
